# rename-by-md5

批量使用文件的 md5 值给文件重命名

```js
[
  "D:\\baidupandownload\\qqqqqqqq.webp",
  "D:\\baidupandownload\\qqqqqqqq-d41d8cd98f00b204e9800998ecf8427e.webp"
];
```
